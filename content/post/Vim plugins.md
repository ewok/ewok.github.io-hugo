Интересные и полезные плагины

* [Follow-my-lead](https://github.com/ktonga/vim-follow-my-lead) - плагин показывает все сочетания из вашего конфига с клавишей leader
* [Gundo](http://sjl.bitbucket.org/gundo.vim) - древовидное UNDO
* [Unite](https://github.com/Shougo/unite.vim) - замена CtrlP и т.д. Быстрый доступ ко всякого рода спискам.
* Fugitive - это git :)
* Syntastic - проверка синтаксиса с очень большим списком поддерживаемых форматов
* [Vim-session](https://github.com/xolox/vim-session) - управление сессиями. Использую для сохранения состояния проектов.
* Surround - это без комментариев :)
* [Vimwiki](https://github.com/vimwiki/vimwiki) - wiki.
